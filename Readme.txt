Python

*******************Steps of Execution*******************
1.) Run new_model_1.py for training the model (adjust the iterations, EPOCHS)
2.) The file saves the model and weights which can be later used(file- model_221.h5, my_model.h5)
3.) For testing on the image Run test_img.py
4.) For testing on the video file Run test_video_file.py  (change the filename in the code)
5.) For testing on the video file Run webcam_video_test.py
